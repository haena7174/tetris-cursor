# 테트리스 (교육용)

HTML, CSS, JavaScript만 사용하는 간단한 테트리스 게임 프로젝트입니다.

## 실행 방법

1. 이 프로젝트 폴더를 연다.
2. `index.html` 파일을 더블클릭하거나, 브라우저로 드래그한다.
3. 게임 화면이 열리면 빈 보드, 점수, 버튼, 조작법 안내가 보인다.

### VS Code / Cursor에서 실행

1. `index.html` 파일을 연다.
2. 우클릭 후 **Open with Live Server** (Live Server 확장이 있는 경우) 또는
3. 파일 탐색기에서 `index.html`을 브라우저로 연다.

> 별도의 설치나 빌드 과정이 필요 없습니다.

## 파일 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 게임 화면 구조 (보드, 점수, 버튼, 조작법) |
| `style.css` | 화면 스타일 |
| `script.js` | 게임 로직 (현재는 빈 보드만 표시) |

## 현재 상태

- 10×20 격자의 빈 게임 보드가 표시됩니다.
- 점수는 `0`으로 시작합니다.
- **시작**, **재시작** 버튼이 있습니다 (게임 로직은 아직 미구현).
- 조작법 안내가 화면에 표시됩니다.
